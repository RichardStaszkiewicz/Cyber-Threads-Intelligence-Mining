A vulnerability ( CVE-2021-29507) was found in a diagnostic log and trace tool of GENIVI Alliance, an automotive industry alliance that develops standard approaches for integrating operating systems and middleware present in the centralized and connected vehicle cockpit. The vulnerability allows a remote attacker to perform a denial of service ( DoS) attack. The vulnerability exists when the configuration file intentionally / unintentionally contains the special characters. A remote attacker can pass specially crafted input to the application and perform a DoS attack.
