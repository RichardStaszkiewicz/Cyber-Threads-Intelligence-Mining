Different capture-replay attacks of key fob signals enable attackers to unlock and start the vehicle. Eavesdropping key fob commands. Engine start command could be gathered. Replaying engine start command. Engine could be started remotely. Replaying engine start command. Engine could be started remotely for one time. After killing the engine the vehicle could not be started again. The continously repeated starting command led to a state in which the vehicle did not accept any other key fob commands.
